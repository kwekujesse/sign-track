# **App Name**: SignTrack

## Core Features:

- Order Entry: Associate scans barcode to capture order number, enters customer name and bin number.
- Customer Search: Customer-facing search by name to find their order.
- Signature Capture: Customer provides a signature to their order, marking it as picked up.
- Associates Dashboard: Dashboard view of all signatures and orders marked as picked up. Generate a end of day pickup report.

## Style Guidelines:

- Primary color: A moderate blue (#c30f16) to instill confidence in the process.
- Background color: A light off-white (#F0F4F5) for a clean, professional feel.
- Accent color: A vibrant green (#388E3C) to highlight key actions and elements.
- Body and headline font: 'Inter', sans-serif, for a clean and modern look that ensures readability on various devices.
- Use simple, clear icons to represent order status, signature completion, and report generation.
- Maintain a clean and intuitive layout with clear separation of sections for order entry, customer search, and the associates' dashboard.
- Subtle animations such as a fading loader upon saving entries and form completion indicators for user feedback.
